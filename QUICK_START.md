# 快速开始指南

## 🚀 三步启动博客

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run server
```

### 3. 访问站点
打开浏览器访问：http://localhost:4000

---

## 📝 常用操作

### 创建新文章
```bash
hexo new "文章标题"
```

### 生成静态文件
```bash
npm run build
```

### 清理缓存
```bash
npm run clean
```

### 部署站点
```bash
npm run deploy
```

---

## 📁 重要目录

- `source/_posts/` - 博客文章
- `themes/defaultone/` - 主题文件
- `public/` - 生成的静态文件
- `_config.yml` - 站点配置
